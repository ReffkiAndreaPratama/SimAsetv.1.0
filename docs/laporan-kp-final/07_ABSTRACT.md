# ABSTRACT

*Asset management at RBTV Bengkulu is still conducted manually using separate written documents and spreadsheet files. This condition causes asset data retrieval to take a considerable amount of time, information on the condition and location of goods is not always accurate, and the preparation of inventory reports cannot be done quickly and efficiently. These problems have the potential to hinder smooth operations and management decision-making related to asset management.*

*This study aims to design and develop a web-based Asset Management Information System at RBTV Bengkulu. Data collection methods used are direct observation and literature study. System analysis is carried out using the PIECES framework to identify weaknesses in the current system. The development method used is the Waterfall model, which includes requirements analysis, system design, implementation, and testing stages. The system is designed using Unified Modeling Language (UML) modeling and developed using the Laravel framework with a MySQL database.*

*The result of this study is a web-based application capable of managing asset data in a centralized manner, including features for asset management, item categories, rooms, QR Codes, maintenance, data import and export, and automatic inventory reports. Based on white-box and black-box testing results, all system functions operate according to user requirements and the system is proven to improve the effectiveness and efficiency of asset management at RBTV Bengkulu.*

**Keywords:** *information system, asset management, Laravel, UML, Waterfall.*
