## Ringkasan Proyek

Proyek ini membangun **Website Pariwisata Berbasis Peta Interaktif** sebagai media promosi destinasi pesisir dan laut Kota Bengkulu. Sistem dikembangkan menggunakan:

- **Backend:** Laravel (PHP Framework)
- **Database:** MySQL
- **Peta Interaktif:** Leaflet.js
- **Metodologi:** SDLC Waterfall
