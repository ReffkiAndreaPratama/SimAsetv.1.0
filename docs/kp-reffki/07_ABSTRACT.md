# ABSTRACT

*Asset management at RBTV Bengkulu is still conducted manually using separate document records and spreadsheet files. This condition causes difficulties in data retrieval, delays in inventory report preparation, and potential recording errors. These issues indicate the need for a computerized system capable of managing asset data in an integrated and efficient manner.*

*This study aims to design and develop a web-based Asset Management Information System at RBTV Bengkulu. The development method used is the Waterfall model, which consists of requirements analysis, system design, implementation, and testing stages. The system is developed using the Laravel framework with a MySQL database. System modeling is carried out using Unified Modeling Language (UML), including Use Case Diagrams, Activity Diagrams, and Class Diagrams.*

*The result of this study is a web-based application capable of managing asset data, asset categories, and asset conditions centrally, as well as generating inventory reports automatically. Based on white-box and black-box testing results, the system operates according to user requirements and helps improve the effectiveness and efficiency of asset management at RBTV Bengkulu.*

**Keywords:** *information system, asset management, Laravel, UML, system testing.*
