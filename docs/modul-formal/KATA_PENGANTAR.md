# KATA PENGANTAR

Puji syukur kehadirat Allah SWT atas rahmat dan karunia-Nya sehingga Modul Pengembangan Sistem Informasi Manajemen Aset Barang Kantor Berbasis Web untuk RBTV Bengkulu ini dapat disusun dan diselesaikan dengan baik.

Modul pengembangan ini disusun sebagai panduan teknis dan praktis dalam membangun, mengimplementasikan, serta mengembangkan sistem informasi manajemen aset berbasis web. Modul ini mengacu pada hasil Kerja Praktik yang telah dilakukan oleh tim pengembang, dengan fokus pada pemanfaatan teknologi web untuk meningkatkan efisiensi pengelolaan aset barang kantor di lingkungan Rakyat Bengkulu Televisi (RBTV).

Secara umum, modul ini membahas tahapan pengembangan sistem mulai dari analisis kebutuhan, perancangan sistem, implementasi hingga pengujian dan evaluasi, dengan pendekatan Software Development Life Cycle (SDLC) model Waterfall. Teknologi yang digunakan dalam modul ini meliputi Laravel sebagai framework backend, MySQL sebagai basis data, serta fitur QR Code untuk identifikasi aset secara cepat dan akurat.

Modul ini diharapkan dapat digunakan oleh:

1. Mahasiswa sebagai panduan praktikum dan pembelajaran pengembangan sistem informasi manajemen aset berbasis web.
2. Dosen sebagai bahan ajar pendukung pada mata kuliah Rekayasa Perangkat Lunak, Sistem Informasi, atau Pengembangan Web.
3. Pengembang dan instansi terkait sebagai referensi teknis dalam pengembangan sistem informasi manajemen aset berbasis web.

Penulis menyadari bahwa modul ini masih memiliki keterbatasan dan membutuhkan pengembangan lebih lanjut, terutama dalam penambahan fitur lanjutan seperti depresiasi aset otomatis, integrasi dengan sistem keuangan, serta pengembangan aplikasi berbasis mobile. Oleh karena itu, kritik dan saran yang membangun sangat diharapkan guna penyempurnaan modul ini di masa mendatang.

Akhir kata, semoga modul pengembangan ini dapat memberikan manfaat dan kontribusi nyata dalam pengembangan sistem informasi manajemen aset berbasis teknologi digital.
